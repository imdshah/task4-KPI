# Task 4 – KPI Dashboard using Tableau

## Dataset Used
- Sample Superstore dataset (Excel)
- Fields used include:
  - Sales
  - Profit
  - Order ID

## KPIs Created
The following Key Performance Indicators (KPIs) were created:

1. **Total Sales KPI**
   - Displays the total sales value.
   - Formatted using Indian currency (₹).

2. **Total Profit KPI**
   - Displays total profit.
   - Highlighted in green to represent positive performance.

3. **Total Orders KPI**
   - Displays the total number of orders using COUNT of Order ID.

4. **Average Order Value (AOV) KPI**
   - Calculated as:
     ```
     SUM(Sales) / COUNT(Order ID)
     ```
   - Represents the average revenue generated per order.

## Tools Used
- Tableau Public (Desktop Edition)
